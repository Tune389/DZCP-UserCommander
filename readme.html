UserCommander - ReadMe

Dieses Addon bietet die möglichkeit User schnell und einfach zu verwalten es lassen sich sämtliche Posts,Beiträge, Nachrichten und Userinformationen auf einer Seite auflisten und auch mit einem Klick löschen

Haftungsausschluss

Wir übernehmen keinerlei Verantwortung, für Schäden die durch das einbinden der Mod/des Addons entstehen. Das einbinden und nutzen erfolgt demnach auf eigene Gefahr.

Vor den einbinden der Mod/des Addons sollte eine Datenbanksicherung durchgeführt werden. Des Weiteren empfehlen wir, von allen Dateien welche geändert werden müssen (unter „Benötigte Dateien“ aufgelistet), ebenfalls eine Sicherheitskopie anzufertigen.
Benötigte Dateien
Folgende Datei werden für diese Mod/dieses Addon benötigt:

  • admin/menue/usercommander.gif
  • inc/additional-languages/deutsch/usercommander.php
  • __template__/admin/usercommander.html
  • __template__/admin/usercommander_msg.html
  • __template__/admin/usercommander_show.html
  • __template__/admin/usercommander_show_main.html
  • __template__/admin/usercommander_show_tr.html
Installation

Ladet alle Dateien Hoch danach ist das Addon einsatzbereit! Macht zur Sicherheit ein Backup der Datenbank!
